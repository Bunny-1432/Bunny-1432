### 1. **🏥 Multi-Disease Prediction System**
**Advanced AI-powered healthcare platform for predicting 50+ diseases**

- **Tech Stack**: FastAPI, React, PyTorch, MongoDB, PostgreSQL, Docker
- **Key Features**:
  - Predicts 50+ diseases across 4 categories (Lifestyle, Chronic, Critical, Dermatological)
  - Multi-modal AI: Supports clinical images, voice analysis, and patient vitals
  - SHAP-based model explainability showing decision drivers
  - AI Medical Chatbot for symptom guidance
  - Full Docker-compose deployment (Frontend, Backend, MongoDB, MLflow)
- **Impact**: End-to-end medical AI system with production-ready architecture
- **GitHub**: [multi-disease-prediction](https://github.com/Bunny-1432/multi-disease-prediction)

### 2. **🎙️ Speech Emotion Recognition System**
**Deep learning system to classify human emotions from audio**

- **Tech Stack**: PyTorch (CNN + LSTM), FastAPI, Next.js, React Three Fiber
- **Key Features**:
  - Hybrid 1D CNN + LSTM architecture for temporal and frequency-domain analysis
  - Extracts MFCC, Chroma, and Mel Spectrogram features from audio
  - FastAPI REST API for real-time inference
  - 3D-enhanced Next.js frontend with dynamic emotion visualization
  - Supports RAVDESS dataset with synthetic data fallback
- **Impact**: Production-ready emotion recognition with sophisticated UI
- **GitHub**: [emotion-recognition-system](https://github.com/Bunny-1432/emotion-recognition-system)

---

### 3. **💳 Credit Risk Scoring Model**
**Machine learning pipeline for predicting credit default risk**

- **Tech Stack**: Python, scikit-learn, Streamlit, pandas, numpy
- **Key Features**:
  - Compares 3 ML models: Logistic Regression, Decision Tree, Random Forest
  - 24 engineered features from 14 raw financial inputs
  - **Results**: Random Forest achieved 0.670 Precision, 0.820 ROC-AUC
  - Interactive Streamlit web dashboard for real-time credit scoring
  - Threshold optimization analysis for deployment
- **Impact**: Production-ready credit risk assessment tool
- **GitHub**: [credit-scoring-model](https://github.com/Bunny-1432/credit-scoring-model)
